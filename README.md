# UEAP_V1 - Analytics Platform

## Projenin Amacı

Bu proje, oyun içi veya genel istatistikleri, oyuncu verilerini ve analizleri izlemek amacıyla geliştirilmiş bir kontrol paneli (dashboard) uygulamasıdır. React Native (Expo) kullanılarak hem web hem de mobil platformlara uyumlu olarak tasarlanmış olup arka planda veri tabanı ve kimlik doğrulama işlemleri için Firebase kullanmaktadır. 

## Teknolojiler
- React Native (Expo)
- Firebase Realtime Database
- Expo Router
- TypeScript
